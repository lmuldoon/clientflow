Professional proposal, payment, and client management for WordPress freelancers and agencies.

== Description ==

ClientFlow gives freelancers and agencies everything they need to send proposals, collect payments, manage projects, and communicate with clients — all from the WordPress admin.

Create professional proposals, collect Stripe payments, manage milestones, automate client communication, and provide a branded client portal directly on your own website.

= Free Features =

* Create up to 3 proposals per month
* Manage clients
* Proposal templates
* Proposal status tracking
* Testimonial request emails
* REST API access
* Freemius-powered upgrades

= Pro Features =

* Everything in Free plus...
* Unlimited proposals
* Stripe payments
* Client portal
* AI writing tools
* Revenue analytics
* Webhooks

= Agency Features =

* Everything in Pro plus...
* Projects & milestones
* Team members
* Client messaging
* File uploads & downloads
* Approval workflows

ClientFlow is designed specifically for WordPress freelancers and agencies who want to manage proposals and client delivery without relying on external SaaS platforms.

== Installation ==

1. Upload the `clientflow` folder to `/wp-content/plugins/`
2. Activate the plugin through the WordPress Plugins menu
3. The setup wizard launches automatically
4. Configure your branding and settings
5. Upgrade to Pro or Agency at any time via ClientFlow → Account

== FAQ ==

= Does ClientFlow support Stripe payments? =

Yes. Stripe payments are available on the Pro and Agency plans.

= Can clients access a portal? =

Yes. The Pro and Agency plans include a branded client portal with magic-link login.

= Does ClientFlow support teams? =

Yes. The Agency plan supports up to 5 team members with role-based access.

= Is multisite supported? =

No. ClientFlow licences are single-site only.

= Does ClientFlow require a third-party SaaS? =

No. ClientFlow runs directly inside WordPress.

= Are AI costs included? =

Yes. AI requests are included with Pro and Agency plans.
